# L2 Telegram Bot — Руководство пользователя

Добро пожаловать!

Эта документация поможет вам установить, настроить и адаптировать L2 Telegram Bot и сопутствующий ему Agent.

Проект распространяется в виде скомпилированных JAR-файлов и конфигурационных файлов. Система состоит из двух частей:

- Hub — автономный Telegram-бот и RMI-сервер, который взаимодействует с Telegram и координирует функции.
- Agent — компонент на стороне игрового сервера, который передает данные в Hub.

### Возможности

Основная цель сервиса предоставление возможности получать события происходящие с персонажем на игровом сервере.

- Гибкая настройка отображаемых ивентов
- Статус персонажа и статистика игровой сессии(полученные дроп, адена)
- Поддержка нескольких серверов
- Локализация интерфейса

### Поддержка нескольких серверов

L2 Telegram Bot разработан на основе архитектуры Hub-Agent, которая поддерживает **одновременную работу с несколькими игровыми серверами**. 

- Один **Hub** может управлять подключениями от нескольких **Agents**.
- Каждый **Agent** соответствует одному игровому серверу (например, Main, X100, Craft‑PvP).
- В Telegram-боте пользователи будут видеть, с каким сервером они взаимодействуют, а администраторы смогут отслеживать все подключенные серверы из одного места.
- **Важно**: Каждый игровой сервер должен иметь уникальный `server.id` в своей конфигурации.

### Быстрый старт

1) Создайте Telegram-бота и получите его токен (см. [Создание Telegram-бота](bot-creation.md)).

2) Отредактируйте `config/hub.properties` и установите `tg.username`, `tg.token` и `tg.admin_chat_ids`.

3) Запустите Hub.

4) Разверните Agent на вашем сервере Lineage II и укажите хост/порт Hub.

5) Откройте Telegram, запустите бота и следуйте меню, чтобы привязать свой аккаунт.
