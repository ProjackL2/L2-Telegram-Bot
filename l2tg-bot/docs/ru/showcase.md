### QR-ссылка сервера
![qr-code-example.png](img/qr-code-example.png){ width="300" }

Agent позволяет генерировать QR-код для ссылки на сервер со всеми необходимыми параметрами для автоматической привязки персонажа к пользователю Telegram.
Игроки могут отсканировать QR-код своим мобильным телефоном, чтобы привязать своего персонажа к пользователю Telegram.

### Запуск бота и успешная привязка
![start.png](img/start.png){ width="300" } ![linked.png](img/linked.png){ width="300" }

Вы можете запустить бота, отправив команду `/start` вашему боту. После успешной привязки персонажа к пользователю Telegram вы можете использовать команду `/menu`, чтобы увидеть меню бота.
Один пользователь может привязать несколько персонажей к одному аккаунту Telegram.

### Статус выбранного персонажа и дроп

Вы можете просмотреть статус выбранного персонажа и выпавшие предметы, выбрав персонажа в меню бота.
