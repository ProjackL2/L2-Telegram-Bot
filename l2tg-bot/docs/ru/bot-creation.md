# Создание Telegram-бота

Это руководство объясняет, как создать Telegram-бота и получить данные, необходимые для работы Hub.

## Создание бота через @BotFather:

1) В Telegram найдите `@BotFather` и начните диалог.

2) Отправьте команду `/newbot` и следуйте подсказкам:

   - Выберите публичное отображаемое имя (его можно будет изменить позже).
   - Выберите уникальное имя пользователя, заканчивающееся на `bot` (например, `L2TestBot`).

3) BotFather ответит вам токеном бота. **Держите его в секрете**.

![img.png](img/new-bot.png)

### Дополнительные настройки BotFather

- `/setdescription`, `/setabouttext`, `/setuserpic` — улучшают профиль вашего бота.
- `/setcommands` — задают подсказки команд (бот также использует меню; команды не обязательны).
- `/setprivacy` — оставьте приватность включенной для личных чатов; отключайте только если понимаете зачем.

Бот поддерживает четыре команды:

 - `/menu` — вызвать главное меню бота
 - `/start`, `/link` — вызвать начальную процедуру
 - `/about` — показать настраиваемую информацию о сервере

Вы можете настроить их отображение в меню бота с помощью команды `/setcommands`. 

![img.png](img/setcommands.png)

## Настройка Hub данными вашего бота

- Откройте `config/hub.properties` на хосте Hub и установите:

```
# Имя пользователя бота (без @)
tg.username=your_bot_name_bot

# Токен от @BotFather (держите в секрете!)
tg.token=1234567890:ABCDEF_your_token_here

# Список ID чатов администраторов через запятую
tg.admin_chat_ids=1111111,2222222
```

### Как узнать свой Telegram Chat ID

- Самый простой способ: откройте `@userinfobot`, нажмите «Старт», и он покажет ваш Chat ID.
- Либо добавьте своего бота в личный чат или группу, отправьте сообщение и используйте любой инструмент или бота для поиска Telegram ID, чтобы узнать ID чата.

## Рекомендации по безопасности

- Никогда не публикуйте токен (включая скриншоты).
- Если токен был скомпрометирован, смените его у @BotFather командой `/revoke`; затем обновите `tg.token` и перезапустите Hub.
- Ограничьте круг администраторов бота с помощью `tg.admin_chat_ids`.

## Следующие шаги

- Вернитесь к разделам [Установка](installation.md) и [Конфигурация](configuration.md), завершите редактирование `hub.properties` и запустите Hub.
- Как только Hub запустится, откройте Telegram и нажмите «Старт» в вашем боте, чтобы начать использование.
