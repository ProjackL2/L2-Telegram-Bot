# Лицензирование

Для работы L2 Telegram Bot Hub требуется действующая лицензия. Лицензия привязана к аппаратному идентификатору вашего сервера (Machine Code).

### Процесс лицензирования

1. **Запуск Hub**: При первом запуске Hub без файла лицензии или с недействительной лицензией работа будет остановлена, а в консоли появится сообщение об ошибке.
2. **Получение Machine Code**: Найдите в консоли строку:
   `Your machine code: ABC123XYZ...`
3. **Передача кода**: Скопируйте этот код и отправьте его разработчику/поддержке.
4. **Получение ключа**: Разработчик сгенерирует лицензионный ключ специально для вашей машины.
5. **Создание файла лицензии**: 
    - Создайте текстовый файл с именем `license.txt` в папке `config/` вашего Hub.
    - Вставьте полученный ключ в этот файл.
    - Убедитесь, что в файле нет лишних пробелов или пустых строк.
6. **Перезапуск Hub**: Снова запустите Hub. Теперь он должен успешно пройти проверку и начать работу.

### Решение проблем
- **Verification Failed**: Если вы видите "LICENSE VERIFICATION FAILED!", убедитесь, что файл `license.txt` находится именно в папке `config/` и содержит точную строку, предоставленную разработчиком.
- **Смена оборудования**: При переносе Hub на другой сервер или значительных изменениях в "железе" ваш Machine Code может измениться, и вам потребуется новый ключ.
